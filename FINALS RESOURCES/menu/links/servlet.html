Java Servlet

  • -  This are Java objects that are intended to components in client server communications.
  • -  It handles clients’ requests by generating responses to such requests.
  • -  It is a part of the JAVA EE specification (see java servlet API for more details).

Java Servlet is an object that hosted in ‘servlet container’.

Introduction

Servlet Lifecycle

  •      - java servlet is used to handle http request & generate http responses.
  •      - this servlet is tied to HTTP Protocol.
  •      - this servlet is used to implement an application’s web layer.
  •      - HTTP Servlet is dependent and used specifically to a HTTP protocol.

  •      - Hosted in ‘web container’ which a component in java ‘application server’ (e.g. Apache TomCat, Red Hat JBoss / WildFly, Oracle GlassFish, IBM Web Sphere)

  •      The service() method call is routed to a doXXX() call, depending on the http request method:
  •          ● doGet()
  •          ● doPost()
  •          ● doHead()
  •          ● doPut()
  •          ● doDelete()
  •          ● doOptions()
  •          ● doTrace()
